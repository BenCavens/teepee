# chief-teepee

## Dependencies:
- Foundation
- Animate.css
- Jquery
- Font-awesome
- Gulp
- Laravel
- Elixir

## Run the application:
1. Open Terminal
2. Git pull for updates/changes
3. bower install kingjohnny/chief-teepee
4. Create your build file (Gruntfile.js and build.scss)
5. Have a Pow wow
